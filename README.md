# globe_app

Overview
Globe App is a cross-platform e-commerce solution that runs on Web, Android, and iOS. It features a beautiful purple dark theme, real-time database synchronization, secure payment processing, and a complete admin panel for product and order management.

Key Highlights
Modern Design - Purple theme with dark mode by default
Real-time Updates - Firestore streams for instant synchronization
M-Pesa Integration - Secure payment processing
Role-based Access - Separate dashboards for users and admins
Cloud Images - Cloudinary CDN for fast image delivery
Cross-platform - Single codebase for web and mobile
